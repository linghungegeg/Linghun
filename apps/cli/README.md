# Linghun

Linghun is a local-first, evidence-first AI coding terminal.

Linghun 是本地优先、证据优先的 AI 编程终端。

```bash
npm install -g @linghun/cli
linghun
```

Full documentation:

- Chinese README: https://github.com/linghungegeg/Linghun/blob/main/README.md
- English README: https://github.com/linghungegeg/Linghun/blob/main/README.en.md
